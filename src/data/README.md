# Portfolio Data

Edit **`portfolio-data.json`** to update your portfolio content in one place.

## What you can update

- **main** – Name, title, email
- **socials** – LinkedIn, GitHub, Instagram URLs (no Twitter)
- **homepage** – Short title/description, **workExperience** (jobs with period, company, position, achievements), **certificates**
- **about** – About title and long description (skills, education)
- **projects** – Array of projects: title, description, logo URL, linkText, link
- **skills** – Array of `{ "category": "...", "items": ["..."] }`
- **contact** – Phone, address, languages

After saving the JSON file, restart or refresh the app to see changes.
